In this week, we are going to build a basic website.

Week 4 - Basic Website

Review Introduction to HTML
  Explain what HTML is and its importance in web development
  Discuss the basic structure of an HTML document, including the head and body sections
  Introduce commonly used HTML tags, such as <html>, <head>, <title>, <body>, <h1>, <p>,   <a>, and <img>
Setting up a Development Environment
  Install a text editor, such as Atom or Sublime Text
  Create a new HTML file and save it with a .html extension
  Open the file in a web browser to ensure it displays correctly
Creating the Page Structure
  Use the <html>, <head>, and <body> tags to structure the page
  Add a <title> tag to give the page a title
  Use <h1> tags to create headings and <p> tags to create paragraphs
  Add an <img> tag to insert an image onto the page
  Use <a> tags to create hyperlinks to other pages or external websites
